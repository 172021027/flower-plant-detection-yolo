# 花草检测项目 - Jupyter Notebooks

本目录包含项目的Jupyter Notebook示例和教程。

## 可用的Notebooks

### 1. 数据探索和可视化
- `01_data_exploration.ipynb` - 数据集探索和统计分析
- `02_data_visualization.ipynb` - 数据可视化示例

### 2. 模型训练和评估
- `03_model_training.ipynb` - 模型训练完整流程
- `04_model_evaluation.ipynb` - 模型性能评估和分析

### 3. 推理和应用
- `05_inference_examples.ipynb` - 推理示例和技巧
- `06_real_time_detection.ipynb` - 实时检测应用

### 4. 高级功能
- `07_custom_training.ipynb` - 自定义训练配置
- `08_model_optimization.ipynb` - 模型优化和部署

## 使用说明

### 启动Jupyter
```bash
# 安装Jupyter
pip install jupyter

# 启动Jupyter服务
jupyter notebook

# 或使用JupyterLab
jupyter lab
```

### 环境要求
确保已安装项目的所有依赖：
```bash
pip install -r requirements.txt
```

### Notebook约定
- 每个Notebook都是自包含的
- 包含详细的中文说明
- 代码可以直接运行
- 包含预期的输出结果

## 创建新的Notebook

如果您想创建新的Notebook：

1. 确保遵循命名约定：`序号_功能描述.ipynb`
2. 在开头添加详细的说明
3. 包含必要的导入和环境设置
4. 添加充分的注释说明
